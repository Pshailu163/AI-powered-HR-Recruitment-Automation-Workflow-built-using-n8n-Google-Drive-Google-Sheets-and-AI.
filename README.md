# AI-Based HR Recruitment Automation using n8n

## Overview

This project automates the recruitment process using n8n workflow automation.

The workflow collects resumes, extracts candidate information, analyzes resumes using AI, scores candidates, filters qualified applicants, stores records, and sends automated emails.

## Features

- Job application collection
- Resume upload automation
- Resume data extraction
- AI-based resume analysis
- Candidate scoring
- Candidate shortlisting
- Google Sheets integration
- Automated email notifications
